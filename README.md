# BUS4005 Prompt Library Portfolio

This repository contains my portfolio for **BUS4005 Assessment 1** at **La Trobe University**.  
The project presents a **10-prompt AI library** designed to support workflow automation in a **retail business environment**, with a particular focus on **HR onboarding, store operations, and management reporting**.

## Project Overview

The purpose of this portfolio is to demonstrate how **structured prompting** can be applied to solve real business problems using AI.

This prompt library was designed as a **consultancy-style solution** for a multi-site retail organisation facing repetitive, high-frequency manual tasks such as:

- onboarding communications
- shift and roster briefings
- training documentation
- employee support responses
- performance reporting
- management insights generation

The library shows how AI can improve:
- efficiency
- consistency
- scalability
- decision support

while still maintaining **human oversight and governance**.

---

## Business Context

**Industry:** Retail  
**Organisation Type:** National multi-site store network  
**Workflow Focus:** HR onboarding, operational communication, and reporting automation

Retail businesses often deal with:
- high employee turnover
- repetitive onboarding activities
- inconsistent communication across store locations
- manual reporting processes
- delayed access to operational insights

This project addresses those issues through a structured set of prompts aligned to real workflow steps.
